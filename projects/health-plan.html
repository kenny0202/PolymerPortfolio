Simple Health Plan

Simple Health Plan is an Android Application that I have developed along with my team Injecting Beans for Introduction to Android Programming course. Our project was to build an application based on a theme called "LifeStyle". We came up with some common ideas such as an exercise planner, a recipe cookbook, and an event planner. We decided to combine the ideas together which allows the user to create their exercises and add them to a workout program or schedule them on a specific date, and also you are able to add recipes and instructions on how to cook the recipe and also plan a meal program. As a student and a developer it may be hard to maintain a healthy lifestyle so we created this application targeting people who want to begin getting back into shape and eating healthy. The application was developed in Java using SQLite as the database system as well as open source libraries.
