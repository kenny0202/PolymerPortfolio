Project Management System

The Project Management System is an enterprise level web application that I have developed along with my team(Private Constructors) for our COMP 4911 Managing Information System Development course. The project management system was built using enterprise level technology to provide high level of security and concurrency. It allowed project managers to manage projects and manage employees.

As a project manager you were also able to generate reports on the current project to allow billing to clients. As a regular employee you are only allowed to enter in time sheets for specific projects that you belong to. Other roles were also involved in this project such as a supervisor who oversees the project as a whole and can assign and remove project managers from a project and a human resource role who can assign employees to a project and approve time sheets that is submitted for approval.

The application was developed using JSF, EJB, Java, JPA, MySQL, PrimeFaces, and Bootstrap.
